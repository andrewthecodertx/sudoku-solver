# Multi-Language Sudoku Solver

Welcome to the Multi-Language Sudoku Solver project! This repository contains Sudoku
solver implementations in various programming languages, allowing you to explore
different language paradigms and compare solver performance.

## Project Overview

The goal of this project is to showcase the versatility of Sudoku solving algorithms
implemented in different programming languages. Whether you're a developer looking
to explore multiple languages or a Sudoku enthusiast interested in the underlying
code, this project provides a unique opportunity to see the same problem solved
across different platforms.

## Supported Languages

- C++
- Go
- Javascript
- PHP
- Python
- Rust

## Features

1. **Unified Interface:** Interact with the Sudoku solver through a common web interface,
making it easy to switch between languages seamlessly.

2. **Live Compilation and Execution:** Compile and run the solver in real-time
directly from the web interface. See the results instantly!

3. **Code Visualization:** Visualize the execution flow of the solver algorithms
in each language, gaining insights into different programming paradigms.

4. **Multi-Language Support:** Comprehensive documentation on how to compile
and run the solver in each language. Sample puzzles are included for testing.

## Getting Started

1. Clone this repository:

    ```bash
    git clone https://github.com/yourusername/sudoku-solver.git
    ```

1. Choose a language folder (e.g., `python`, `rust`) and follow the
instructions in the respective README to compile and run the solver.

1. Visit the web interface (`index.html`) to interact with the solver using
your preferred language.
